## Deployment

### Frontend

Open your terminal and navigate to the frontend directory

```bash
    cd frontend
```

(Optional) Copy the example environment file to create your own:

```bash
    cp .env.example .env
```

Install the required dependencies:

```bash
    npm install
```

Start the frontend server:

```bash
    npm start
```
